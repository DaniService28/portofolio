# Project Title (Update)
This project is a dynamic portfolio web application built using Node.js, Express, and EJS. It showcases personal information, skills, and projects through a clean dashboard-style interface. The site also includes a fully functional contact form that validates user input and sends real emails using Nodemailer. All pages are generated dynamically on the server, and the design uses custom CSS and images stored within the project.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Install dependencies using npm install
2. Create a .env file with email credentials for Nodemailer
3. Run the server with node app.js

Instructions for using the software:

1. Open the browser and navigate to http://localhost:3000
2. Explore the dashboard sections (Profile, Skills, Projects, Contact)
3. Use the contact form to send a real message to the configured emai


## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:
* Node.js (v18 or later)
* Express (v4)
* EJS (v3)
* Nodemailer (v6)
